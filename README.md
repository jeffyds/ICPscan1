# ICPscan  
通过域名或备案号，查询备案信息中的所有域名  

## Install  
去release中下载

## Notice
云厂商VPS使用本工具可能会导致失效，需要手工检查访问https://icplishi.com/是否会跳转https://sewer.ip138.com/

## Usage  
./ICPscan_amd64 -h  
Usage:  

Flags:  
  -f, --file string   从文件中读取目标地址 (Input FILENAME)  
  -h, --help          help for ICPscan  
  -u, --url string    输入目标地址 (Input IP/DOMAIN/URL)
